Our Operating System

Methuselah OS is an in-developement Linux distribution that strives
for ease-of-use and universal applications such as
package installers, specialised command correctors and more!

Methuselah also has a custom neofetch and it's own in-developement
KDE based upon the breeze KDE. The first public alpha of Manjaro
{Methuselah OS Public Alpha Release 0.2 Jupiter MB (Manjaro Basis)} is soon to
be released so stay tuned into the website and check in every
now and then to discover if it has been released.
