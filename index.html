Our Operating System

Methuselah OS is an in-developement Linux distribution that strives
for ease-of-use and universal applications such as
package installers, specialised command correctors and more!

Technical Side Of Things

The first version of Methuselah OS will be CLI based and not have a desktop environment.
But what sets Methuselah OS apart from the other linux distros? Great question! Certain applications are restricted to a certain package manager,
such as pacman or yum. We intend on including all open source package installers to rid of this unessecary restriction.
